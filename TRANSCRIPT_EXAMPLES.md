# Meeting Transcript Examples

Use these examples to test the Aggroso action item extraction. Just copy and paste the text into the app.

## Example 1: Project Sync
**Transcript:**
Aman: Okay, let's start the weekly sync. Rohit, how is the login page coming along?
Rohit: The UI is done. I'm working on the backend integration now. I should be able to finish it by this Thursday.
Aman: Sounds good. Neha, did you finish the documentation for the API?
Neha: Not yet, I was caught up with some bugs. I'll get the API docs ready by tomorrow evening.
Aman: Okay. Also, we need to schedule a meeting with the design team. Neha, can you handle that?
Neha: Sure, I'll reach out to them today and set something up for Monday.
Rohit: I also need the updated assets for the dashboard.
Aman: I'll talk to the designer and send those over to you by EOD.

## Example 2: Client Kickoff
**Transcript:**
Manager: Welcome everyone to the Project Alpha kickoff. Our main goal for this week is to finalize the requirements.
David: I've started the draft. I'll share the document with the team by 2 PM today for feedback.
Manager: Thanks David. Sarah, what about the staging environment setup?
Sarah: I'll handle the AWS configuration. It will be ready for deployment by Wednesday morning.
Client: We also need to see the initial wireframes.
Sarah: Our designer will have those ready for your review by next Friday.
Manager: Perfect. I'll send out the meeting minutes after this call.

## Example 3: Error Post-Mortem
**Transcript:**
Kevin: The database crash yesterday was due to a memory leak in the reporting service.
Priya: We need to fix this immediately. Kevin, can you investigate the leak today?
Kevin: Yes, I'm on it. I'll have a fix ready for testing by tomorrow.
Priya: Good. I'll monitor the server logs in the meantime to catch any spikes.
Kevin: We should also update our alert thresholds.
Priya: Agreed. I'll adjust the monitoring settings on Friday once the fix is deployed.
